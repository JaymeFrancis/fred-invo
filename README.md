## Fred-Invo Asset and Auto Supply Management System

This system is used for keeping track of the inventory and sales of the Automobil Shop. The tech stack that used for the development of the system is TALL Stack [TailwindCSS, AlpineJS, Laravel, and Livewire]. The system is still under development.
